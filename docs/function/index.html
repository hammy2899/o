    public sort(object: object, iterator: function(a: object, b: object), follow: boolean): object version 1.0.0 since 1.0.0 source 

    import sort from 'o/src/sort.js'

    Sort an object via the iterator evaluation

    Params:

    NameTypeAttributeDescription
    object object

    The object to sort
    iterator function(a: object, b: object)

    The function to evaluate
    follow boolean
    • optional
    • default: false

    Whether to follow objects

    Return:

    object

    The sorted object

    Example:

    const a = { a: 4, b: 7, c: 3 };
sort(a, (a, b) => {

 if (a.value > b.value) return 1;
 return 0;
}); // => { c: 3, a: 4, b: 7 }
